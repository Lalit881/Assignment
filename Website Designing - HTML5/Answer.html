Website Designing - HTML5

Q.1 ) What are the new tags added in HTML5?

<article> : Represents a self-contained piece of content that can be independently distributed or reused, such as blog posts, articles, and forum posts.
<section> : Defines a thematic grouping within a document. It's often used to group related content together.
<nav>: Signifies a section of the page that contains navigation links, such as menus or table of contents.
<header>: Represents the introductory content of a section or page. It usually contains headings, logos, and navigation elements.
<footer>: Represents the closing content of a section or page, often containing copyright information, author details, and links to related content.
<aside>: Used for content that is related to the main content but can be considered separate from it, like sidebars, pull quotes, or advertisements.

Q.2 ) How to embed audio and video in a webpage?

1. Save your audio and video files in a new directory on your computer.
2. Create a new HTML file in the same directory, called index.html.
3. Add <audio> and <video> elements to the page; make them display the default browser controls.
4. Give both of them <source> elements so that browsers will find the audio format they support best and load it. These should include type attributes.
5. Give the <video> element a poster that will be displayed before the video starts to be played. Have fun creating your own poster graphic.

Q.4 ) Canvas and SVG tags?

canvas

1. The <canvas> element provides a drawing surface through JavaScript. It allows for the creation of dynamic, bitmap-based graphics.
2. Canvas requires scripting (usually JavaScript) to draw shapes, images, and other graphical elements.
3. It is essentially a blank slate, and you use JavaScript to draw on it.
4. Canvas is well-suited for applications that require real-time rendering, such as games or interactive data visualization.
5. Graphics drawn on a canvas are not retained as objects; once drawn, they are pixels on a bitmap and can't be individually manipulated.
6. Canvas graphics are resolution-dependent, meaning they can appear pixelated on high-resolution displays.

SVG

1. SVG (Scalable Vector Graphics) is an XML-based vector image format that allows for the creation of resolution-independent graphics.
2. SVG is more like a document structure, similar to HTML, and uses XML tags to define shapes, paths, and other graphical elements.
3. It's easy to manipulate and animate SVG elements using CSS or JavaScript.
Graphics in SVG are retained as objects, which means 4. they can be manipulated individually (e.g., scaling, rotating).
5. SVG graphics are resolution-independent and scale perfectly on all devices and screen sizes.
